# AI Research Paper Intelligent System

## Overview

The AI Research Paper Intelligent System is a Natural Language Processing (NLP) project developed using Python and Google Colab. The system retrieves research papers from the arXiv repository based on a user-provided topic and analyzes the selected paper using various NLP techniques.

## Features

- Research Paper Search using arXiv API
- Research Paper Summarization
- Keyword Extraction
- Named Entity Recognition (NER)
- Research Category Detection
- Similar Research Paper Recommendation

## Technologies Used

- Python
- Google Colab
- arXiv API
- spaCy
- NLTK
- Sumy
- YAKE

## Project Workflow

1. Enter a research topic.
2. Search research papers from arXiv.
3. Display the top research papers.
4. Select a research paper.
5. Generate a summary of the abstract.
6. Extract important keywords.
7. Perform Named Entity Recognition (NER).
8. Predict the research category.
9. Recommend similar research papers.
10. Display the PDF link.

## Data Source

The project retrieves research papers dynamically from the arXiv API. No local dataset is required.

Website: https://arxiv.org/
